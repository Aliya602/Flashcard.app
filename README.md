# Flashcard.app 🎴
A Python flashcard application (command line) with multiple choice support, save/load functionality, and tracking your score

## Features/Functionality
- Create custom decks of flashcards
- Multiple choice support with shuffle options
- Save decks to JSON files
-  Load previously saved decks
-  Score tracking with percentage (%)
-  Personalized encouragement message / feedabck
-  Colorful terminal interface

## How to run  code:
- Run python code

## Requirement
- Python 3.x
- colorama (pip install colorama)

## Instructions
1. Create a new deck or load an existing one
2. Add flashcards with optional wrong answers for multiple choice
3. Save your deck
4. Take a quiz and track your score
